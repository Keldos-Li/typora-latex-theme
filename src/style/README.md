## 设置说明
- _setting 总设置
- 段前距 = margin-top
- 段后距 = margin-bottom
- 段前段后距总体上使用 margin 控制, word 的段前段后距效果和 css 的 margin 一样会[外边距重叠](https://developer.mozilla.org/zh-CN/docs/Web/CSS/CSS_Box_Model/Mastering_margin_collapsing).
